# LED-Illumination
A teaching material for event attendees to learn soldering through hands-on experience

# LEDイルミネーションキット

初級用のはんだ付け教材を想定した電子工作キットです。

- 地域イベントなどで、はんだ付け体験（ハンズオン）に利用できます。
- 短時間とするため組み立てはリード線の抵抗3本、はんだ付けは計6個所のみです。
- リード線の折り曲げ、基板への挿入、はんだ付け、切断の一連の作業を体験できます。


## 動作
- ボタンを押すと9個のLEDが点滅します。
- 点滅しない場合、一旦ボタンを離して再度押すと点滅することが多いです。

## 回路

 [回路図](./bf-045_sch.pdf)

- 3段の無安定マルチバイブレータです。
- ボタン押下(電源オン)で発振する確率を上げるため抵抗値に変化を付けました。
- LED駆動用のトランジスタを追加しています。
    - 3V電池でLEDを十分点灯させる
    - 3組のLEDうち1組のみ消灯させる
- LEDの電流制限抵抗値は、現物に合わせました。
    - JLCPCBのPCBAにおいてBASIC部品のLEDを選択
    - 実際に点灯させて、赤緑黄の各色の明るさが同程度になる抵抗値を選択
